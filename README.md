Sales_DB - Consultas SQL aplicadas ao mundo real

Este projeto não tem foco em criar banco de dados via código.
O banco foi construído em Excel, refinado e depois importado para o SQL Server.
O objetivo é demonstrar domínio prático de SQL, usando consultas que respondem a problemas reais de vendas, marketing, gestão e operações.

Objetivo do Projeto

Entregar um conjunto sólido de consultas SQL que demonstram domínio técnico e pensamento analítico aplicado ao contexto de vendas. O projeto cobre:
consultas básicas utilizadas no dia a dia,
joins e análise entre múltiplas tabelas,
agregações relevantes ao negócio,
window functions para análises mais avançadas,
perguntas estratégicas de negócio respondidas via SQL.

Estrutura do Banco de Dados

O banco contém 5 tabelas principais:
Products
Customers
Stores
Employees
Sales
